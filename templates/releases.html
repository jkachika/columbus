    Columbus v1.0.0-Alpha

    This is a pre-production release. You can download the source code below.

    Download zip      Download tar.gz

    Features

    • Creation of components, combiners, and workflows
    • Sharing the workflows with other users
    • Visualizing components and combiners with interested parties
    • Creation of constraints
    • Bigquery or Combiner as data source for workflows
    • Starting single workflows with 'for' and multiple workflows with 'foreach'
    • Combining workflows with in specific time range
    • Supported output types include CSV List, Feature, Feature Collection, Multi Collection and Generic List.
    • History of all workflows run by a user
    • Downloading data for Feature Collection and Multi Collection
    • Read access to the visualization files in Google Drive
    • Deletion of components, combiners and workflows
    • Deletion of workflow runs in History
    • v1.0.0-alpha
